Description

This goal of this project was to build a javaFX application that implemented the iTunes Search API to grab information based on the users search query. It would then display the top 20 most popular album covers that contain the users search tag. The user could also hit the play button to cyclically replace images with others within the top 40 results.

Technologies Used

  • Java
  • JavaFX
  • iTunes Search API
  • Eclipse
  • Maven

Learning Outcomes

  • Learned how to implement an API and how to parse information between user submitted form data and data returned to the system through the API.
  • Gained an initial understanding of Java GUI development using javaFX.
  • Expanded my ability to independently research and examine documentation on a technology or API that I wasn't exposed to before.

Description

My first personal project (outside of the Iris Dataset) that I worked on while learning Data Science/Machine Learning to accurately predict who survived the historical event of the sinking of the Titanic. The current model provides an accuracy of 78.2%, which ranks ~82nd percentile in the Kaggle Competition. The CSV output files and corresponding code are used to create accuracy checks on Kaggle.com.

Technologies Used

  • Jupyter Notebook
  • Python
  • KNN
  • SVC
  • Seaborn
  • Kaggle

Learning Outcomes

  • Ability to creatively analyze a given dataset to optimize the success rate of your algorithms.
  • Efficient preprocessing of data with the removal/replacement of NULL data
  • Establish a clear and concise visual representation of what the data is describing.
  • Learned powerful machine learning algorithms such as SVC and KNN to build an accurate prediction model.

This Project was part of Univeristy of Georgia's CSCI 1302 Class. The sharing or publication of the source code for this project is against UGAs Code of Conduct.
