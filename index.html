Description

This project is the portfolio website you are viewing right now! It was a personal project I thought would be fun to work on during winter break of 2019 on into early 2020. I used vanilla Javascript along with parts of jQuery for the functionality of this website. I also used bootstrap to implement the carousel that I worked on from the E-Cinema Booking project.

Technologies Used

  • HTML
  • CSS
  • Javascript
  • jQuery
  • Bootstrap
  • Github Pages
  • Google Domains

Learning Outcomes

  • This is the first project that I hosted online to the public. So I learned more about purchasing and working with domains, along with implementing GitHub Pages.
  • This project helped me learn the importance of planning in the development and deployment of this website, as it was the first major web project that I undertook alone and had full control.
